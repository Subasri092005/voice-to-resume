🔧 JAVASCRIPT BEHAVIOR:

  1. Page loads → initConversation()
  2. Fetch /conversation/init → Get greeting
  3. Speak greeting via speechSynthesis
  4. Auto-start listening (SpeechRecognition)
  5. User speaks → Transcript captured
  6. Add user message to chat (right bubble)
  7. POST /conversation/submit with answer
  8. Get next question from backend
  9. Add AI message to chat (left bubble + speaker)
  10. Speak question automatically
  11. Auto-listen for next answer
  12. Repeat until conversation complete

Run this:
  $ python app.py

Then visit:
  http://127.0.0.1:5000/assistant

The assistant will:
  1. Load elegant chat interface
  2. Speak greeting automatically
  3. Auto-activate microphone
  4. Listen for user's answer
  5. Display response in right chat bubble
  6. Show AI response in left chat bubble with speaker icon
  7. Update sidebar progress indicator
  8. Repeat for each question (no clicking needed)
  9. Feel premium and conversational (like Lovable.ai)
